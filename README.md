# 🤔 Navegação Consciente v2.0

Uma extensão do Chrome que te ajuda a navegar de forma mais consciente, com timer de reflexão e lista personalizada de sites.

## Novidades da v2.0

- ✅ **Lista personalizada** - Escolha quais sites terão o aviso
- ⏱️ **Timer de reflexão** - Configure de 3 a 30 segundos de espera
- 🎨 **Interface melhorada** - Visual mais bonito e intuitivo
- ➕ **Adicionar site rápido** - Um clique para adicionar o site atual

## Como instalar

1. Abra o Chrome e vá para `chrome://extensions/`
2. Ative o **Modo do desenvolvedor** (canto superior direito)
3. Clique em **Carregar sem compactação**
4. Selecione a pasta `extensao-impulso-v2`
5. Pronto! A extensão está instalada

## Como usar

1. **Clique no ícone** da extensão na barra do Chrome
2. **Adicione sites** que você quer monitorar (ex: youtube.com, twitter.com)
3. **Configure o timer** (quanto tempo precisa esperar antes de entrar)
4. Ao visitar um site da lista, o modal aparecerá com o timer

## Dicas de uso

Esta extensão é ótima para:
- 🎬 Redes sociais (YouTube, Instagram, TikTok, Twitter)
- 🎮 Sites de jogos
- 🛒 Lojas online (evitar compras por impulso)
- 📰 Sites de notícias
